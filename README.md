### [Rivetr](https://github.com/KwaminaWhyte/rivetr)

A fast, lightweight **deployment engine** built in Rust. Single-binary PaaS for deploying apps from Git with minimal resource usage.

**Features:**
- Git webhook deployments
- Embedded reverse proxy with auto HTTPS (Let's Encrypt)
- Web dashboard for deployment management
- Real-time log streaming
- Uses only 30-80MB RAM (vs 400-800MB alternatives)
